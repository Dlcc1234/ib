Dans plusieurs ateliers, on indique de fermer un outil puis d'en agrandir la fenêtre par la suite : soit indiquer d'iconiser les outils au lieu de les fermer, soit de relancer les outils ensuite.

Démarrage des VMs

Pour s'assurer que le démarrage des VM se passe correctement, on pourra utiliser les commandes powershell suivantes :
  • sur SEA-DC1 :
    • Powershell.exe
    • get-netAdapter|restart-netAdapter
    • restart-computer
  • sur lon-svr1, sea-svr1, sea-svr2 et sea-cl1 (dans cet ordre) : restart-computer
