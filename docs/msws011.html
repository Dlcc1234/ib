remarque ateliers

Dans plusieurs ateliers, on indique de fermer un outil puis d'en agrandir la fenêtre par la suite : soit indiquer d'iconiser les outils au lieu de les fermer, soit de relancer les outils ensuite. Pour faciliter les démonstrations, le script "iscsiCluster.ps1" lançé sur sea-adm1 met en place les 3 disques partagés iSCSI nécessaires dans l'atelier/les démonstrations.

Nettoyer les ateliers

Le code suivant permettra de s'assurer que l'environnement d'atelier a bien démarré (à lancer en powershell depuis SEA-DC1 après que toutes les machines du lab aient démarré) : Powershell Get-netAdapter|restart-netAdapter;Get-ADDomain|foreach-object {get-ADComputer -Filter * -searchBase $_.computersContainer|foreach-object {echo "Redémarrage de $($_.DNSHostName)";Restart-Computer -ComputerName $_.DNSHostName -Force}}

SSO sur Windows Admin Center

Le code suivant permet d'activer le SSO sur Windows Admin Center : Powershell $wacobject = Get-ADComputer -Identity $env:computername; Get-ADDomain|foreach-object { get-ADComputer -Filter * -searchBase $_.computersContainer|foreach-object {Set-ADComputer -Identity $_.Name -PrincipalsAllowedToDelegateToAccount $wacobject;Invoke-Command -ComputerName $_.name -ScriptBlock {klist purge -li 0x3e7}}}

Désactiver le firewall Windows

Le code suivant permet de simplement désactiver le firewall Windows sur un serveur (y-compris core) Powershell Set-NetFirewallProfile -Profile Domain,Public,Private -Enabled False
