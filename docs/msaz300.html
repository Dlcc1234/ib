Un dossier "Ateliers MSAZ300" a été déposé sur le bureau. Il contient surtout:
  • Les fichiers nécessaires à la réalisation des ateliers
  • Un lien vers le site de création des compte azure basé sur les codes fournis pour la session, nommé Azure - Validation pass
  • Un lien vers les instructions des ateliers sur Github
Le Framework .Net 4.8 a été installé sur chaque poste, Le module PowerShell AZ a été installé dans sa dernière version. Le dossier Documents contient (éléments rajoutés par la commande ibSetup MSAZ103 -trainer):
  • Une présentation PowerPoint "AZ-300AIntro" pour aider à l'introduction du stage,
  • Une présentation PowerPoint "AZ-103APrequel" pour aider à l'animation du début du stage,
Dés le début du stage, faire créer un compte Microsoft aux stagiaires (https://www.live.com) Ce compte pourra ensuite être utilisé pour créer un compte Azure de test (raccourci Azure - Validation pass). Vidéo de démonstration de la mise en place d'une connexion vNet to vNet Pour fluidifier le déroulement du stage, ne pas hésiter à conseiller aux stagiaires de réaliser l'(éventuel) exercice 0 de chaque atelier au lancement de la présentation théorique.
