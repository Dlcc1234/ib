MSAZ900 - Accompagnement Pédagogique et Technique de stage

Dernière édition le 19 Juin 2019

Customisation spécifique du stage MSAZ900

Un dossier "Manipulations MSAZ900" a été déposé sur le bureau. Il contient :
  • Les fichiers nécessaires à la réalisation des ateliers (dans des sous-dossiers nommés AZ-100.x / AZ-101.x)
  • Un lien vers la création de comptes Azure gratuits (nécessite de saisir un numéro de carte bancaire).
  • Un lien vers le portail Azure
  • Un lien vers le Cloud Shell Azure
Le Framework .Net 4.8 a été installé sur chaque poste,
Le module PowerShell AZ a été installé dans sa dernière version.
Le dossier Documents contient (fichier rajouté par la commande ibSetup MSAZ103 -trainer) une présentation PowerPoint "AZ-900AIntro" pour aider à l'animation du début du stage.

Conseils d'animation pour le stage MSAZ900

Ce stage ne contenant pas de Pass Azure, conseiller aux stagiaires de se crér un compte Microsoft (sur https://www.live.com) et de se créer un compte Azure gratuit.
La création d'un compte Azure gratuit, hors Pass Azure, nécessite de saisir des inforamtions de Carte Bancaire : prendre le temps de dédramatiser la démarche, en expliquant que la carte ne sera jamais débitée, que si le compte Azure venait à dépasser la volumétrie de services offerte, c'est le compte qui serait désactivé.
Prendre connaissance du fichier readme posé sur le bureau et suggérer aux stagiaires de faire de même.
En cas de soucis sur un poste, il est possible de (re)lancer la commande ibSetup MSAZ900 -force pour mettre en place ou corriger les éléments de customisation décrits précédemment.
(y ajouter l'option "-trainer" sur le poste du formateur).
Le but de la journée étant de faire une introdcution à Azure, elle va s'avérer très théorique : Ne pas hésiter à proposer régulièrement de faire des démonstrations si le temps le permet et inviter les stagiaires à se replonger par la suite dans les vidéos et les walkthrough présents dans leur support.
Le service technique ib est à votre disposition pour tout assistance pendant la durée de la formation.
