MSAZ100 - Accompagnement Pédagogique et Technique de stage

Dernière édition le 19 Juin 2019

Customisation spécifique du stage MSAZ100

Un dossier "Ateliers MSAZ100" a été déposé sur le bureau. Il contient surtout:
  • Les fichiers nécessaires à la réalisation des ateliers
  • Un lien vers le site de création des compte azure basé sur les codes fournis pour la session, nommé Azure - Validation pass
  • Un lien vers les instructions des ateliers sur Github
Le fichier readMe sur le bureau contient, pour référence éventuelle, le titre de chaque atelier, avec l'ordre conseillé de réalisation de ceux-ci, correspondant à la numérotation des modules.
Le module PowerShell AzureRM a été installé en version 6.12.
Le dossier Documents contient (éléments rajoutés par la commande ibSetup MSAZ103 -trainer):
  • Une présentation PowerPoint "AZ-100AIntro" pour aider à l'animation du début du stage,
  • Une présentation PowerPoint "AZ-100A-Extra-Automation" (extraite de l'ancien stage Azure M20533) pour ajouter la thématique "Azure Automation" si demande des stagiaires.

Conseils d'animation pour le stage MSAZ100

Dés le début du stage, faire créer un compte Micrsoft aux stagiaires (https://www.live.com)
Ce compte pourra ensuite être utilisé pour créer un compte Azure de test (raccourci Azure - Validation pass).
Prendre connaissance du fichier readme posé sur le bureau et suggérer aux stagiaires de faire de même.
En cas de soucis sur un poste, il est possible de (re)lancer la commande ibSetup MSAZ100 -force pour mettre en place ou corriger les éléments de customisation décrits précédemment.
(y ajouter l'option "-trainer" sur le poste du formateur).
Le service technique ib est à votre disposition pour tout assistance pendant la durée de la formation.
