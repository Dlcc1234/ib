Conseils génériques concernant Azure:

Indiquer que, pour fluidifier l'expérience, les manipulation reposant purement sur le portail web n'ont pas besoin d'être réalisées à distance. Dans la situation actuelle (crise des Pass Azure):
  • il faudra fournir un tenant ou un resource group aux stagiaires.
  • Dans le cas d'un resource Group, les instructions d'atelier devront être adaptée pour utiliser le RG fourni (et non celui créé dans le lab).
  • Les machines "Standard_A2" semblent être plus disponibles, en région "West US"
Vidéo de démonstration de la mise en place d'une connexion vNet to vNet Pour fluidifier le déroulement du stage, ne pas hésiter à conseiller aux stagiaires de réaliser l'(éventuel) exercice 0 (ou autre exercice de préparation) de chaque atelier au lancement de la présentation théorique.
