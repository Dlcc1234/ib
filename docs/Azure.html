Conseils génériques concernant Azure:

Indiquer que, pour fluidifier l'expérience, les manipulation reposant purement sur le portail web n'ont pas besoin d'être réalisées à distance. Dés le début du stage, faire créer un compte Microsoft aux stagiaires (https://www.live.com) Ce compte pourra ensuite être utilisé pour créer un compte Azure de test (raccourci Azure - Validation pass). Vidéo de démonstration de la mise en place d'une connexion vNet to vNet Pour fluidifier le déroulement du stage, ne pas hésiter à conseiller aux stagiaires de réaliser l'(éventuel) exercice 0 (ou autre exercice de préparation) de chaque atelier au lancement de la présentation théorique. Nota: Pour faire une démonstration impliquant la mise en place de l'autoscale, commencez par passer la commande Powershell suivante dans le Cloud Shell: Register-AzResourceProvider -ProviderNamespace 'microsoft.insights'
