Un dossier "Ateliers MSAZ301" a été déposé sur le bureau. Il contient surtout:
  • Les fichiers nécessaires à la réalisation des ateliers
  • Un lien vers le site de création des compte azure basé sur les codes fournis pour la session, nommé Azure - Validation pass
  • Un lien vers les instructions des ateliers sur Github
Le Framework .Net 4.8 a été installé sur chaque poste, Le module PowerShell AZ a été installé dans sa dernière version. Le dossier Documents contient (éléments rajoutés par la commande ibSetup MSAZ301 -trainer):
  • Une présentation PowerPoint "AZ-301AIntro" pour aider à l'introduction du stage,
  • Une présentation PowerPoint "AZ-103APrequel" pour aider à l'animation du début du stage,
Dés le début du stage, faire créer un compte Microsoft aux stagiaires (https://www.live.com) Ce compte pourra ensuite être utilisé pour créer un compte Azure de test (raccourci Azure - Validation pass). Les ateliers sur la plateforme goDeploy sont largement obsolètes, préférer jouer les ateliers depuis les instructions Github.
